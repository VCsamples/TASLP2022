There are 7 experimental systems to be compared:
(1) BNF2Wav : The baseline system that converts input BNFs directly into a speech waveform.
(2) BNF2Wav(F0) : This system is an extension to BNF2Wav. It takes F0 as an additional input.
(3) BNF2Mel-BNF : This system converts BFNs into Mel-spectrogram, and reconstructs the speech waveform using the PWG vocoder. The frame-wise BNF loss is applied to the generated Mel-spectrogram.
(4) BNF2Wav-BNF : This is similar to the BNF2Mel- BNF system but without involving Mel-spectrogram. The frame-wise BNF loss is applied to the generated waveform.
(5) BNF2Wav-ASR : This system is similar to the BNF2Wav-BNF system except that the segmetal-level ASR loss is used instead of BNF loss, applied to the generated waveform.
(6) BNF2Wav-BNF-ASR : This is the proposed XVC system with two linguistic losses.
(7) BNF2Wav-BNF-ASR(F0) : This system adds the extra F0 to the input, the rest is similar to the BNF2Wav-BNF-ASR system.
